2048 – Игра в терминале на 7 языках
Классическая головоломка 2048 прямо в вашем терминале!
Свайпайте плитки, объединяйте их и стремитесь к заветной цифре 2048.
Реализована на семи языках программирования с единым интерфейсом, подсчётом очков, системой рекордов, отменой ходов, анимацией и настраиваемой сложностью.

🚀 Возможности
Классический геймплей – объединяйте одинаковые плитки, чтобы получить 2048 и выше.

Управление – клавиши WASD для перемещения, Q для выхода, U для отмены последнего хода.

Подсчёт очков – текущий счёт и лучший результат (сохраняется в файл).

Отмена хода – возможность вернуть одно действие назад (полезно при ошибочных ходах).

Уровни сложности – размер поля 4×4 (классика), 5×5, 6×6 (для опытных игроков).

Цветная анимация – плитки подсвечиваются разными цветами в зависимости от значения.

Автосохранение – состояние игры и рекорд сохраняются при выходе.

Статистика – количество ходов, время игры (в некоторых версиях).

Кроссплатформенность – работает на Windows, Linux, macOS.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [опции]
Опции
Опция	Описание
-s, --size <N>	Размер поля: 4, 5 или 6 (по умолчанию 4)
-h, --help	Справка
Управление
Клавиша	Действие
W / ↑	Сдвинуть плитки вверх
A / ←	Сдвинуть плитки влево
S / ↓	Сдвинуть плитки вниз
D / →	Сдвинуть плитки вправо
U	Отменить последний ход
Q	Выход
Примеры
bash
# Классическая игра 4×4
python 2048.py

# Игра на поле 5×5
python 2048.py -s 5

# Игра на поле 6×6
python 2048.py --size 6

🧠 Логика игры
Поле – матрица N×N, изначально заполнена двумя случайными плитками (2 или 4).

Ход – игрок нажимает клавишу направления, все плитки сдвигаются в этом направлении, одинаковые объединяются.

Новая плитка – после каждого хода на пустую клетку появляется новая плитка (2 или 4).

Победа – когда появляется плитка со значением 2048 (или выше для больших полей).

Поражение – когда нет свободных клеток и нет возможных объединений.

Счёт – сумма всех объединённых плиток.

Рекорд – максимальный счёт за все игры, сохраняется в файл.

✨ Дополнительные фичи
Отмена хода – история последних ходов позволяет откатиться на один шаг.

Рекорды – сохраняются для каждого размера поля отдельно.

Цветовая схема – каждая плитка имеет свой уникальный цвет.

Автосохранение – при выходе игра сохраняется, можно продолжить позже.
